## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed.

```sh
npm install npm@latest -g
```
### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/jaws-1684/memory-card.git
   ```
2. Navigate into the project directory
   ```sh
   cd memory-card
   ```
3. Install dependencies
   ```sh
   npm install
   ```


---

## Usage

### Development server
```sh
npm run dev
```
Opens a live-reloading dev server powered by vite.

### Production build
```sh
npm run build
```
Outputs compiled assets into the `dist/` folder.

### Run tests
```sh
npm run test
```

### Run linter
```sh
npm run lint
```
